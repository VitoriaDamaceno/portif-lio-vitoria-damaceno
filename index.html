Projetos de Análises de Dados

Todos os projetos foram desenvolvidos durante as aulas como forma de prática e aperfeiçoamento das habilidades com Power BI.

Projeto 1: Aula 10.09

Dashboard criado com base em dados fornecidos em aula. Explora conceitos básicos de visualização, segmentação e gráficos interativos.

Projeto 2: Colunas Personalizadas

Projeto com foco em manipulação e criação de colunas personalizadas usando DAX, destacando KPIs e dados transformados com lógica de negócio.

Projeto 3: Dashboard AutoSolutions

Dashboard desenvolvido para uma empresa fictícia do setor automotivo. Análise de vendas, serviços, performance de atendimento e outros indicadores importantes.
